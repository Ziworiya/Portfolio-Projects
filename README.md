# Portfolio-Projects
Macroeconomic Impact Analysis: 2026 Geopolitical Crisis 

Objective: Developed a Python-based tracking system to analyze the correlation between energy prices and safe-haven assets during the 2026 Iran conflict. 

Methodology: Utilized the YFinance API to scrape real-time market data; performed data cleaning and normalization using Pandas. 

Key Insight: Identified a "liquidity-driven decoupling" where Brent Crude surged 20% while Gold prices dropped 9%, providing data-driven rationale for shift in market volatility. 

Outcome: Automated visualization reports using Matplotlib to model the impact of energy shocks on regional inflation.
